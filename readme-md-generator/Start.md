# readme-md-generator 使用总结

## 安装

```
npm -g i readme-md-generator
```

## 使用

在项目根目录执行如下`readme`命令

## 总结

使用`readme-md-generator`来创建`README.md`文件方便、规范。但也存在不够灵活问题，比较适用于创建`npm`的包`README.md`文档。
